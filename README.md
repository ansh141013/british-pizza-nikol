# British Pizza Nikol – Ahmedabad

Ahmedabad's favourite unlimited pizza & salad buffet website. 51+ varieties of freshly baked pizzas, live salad bar, soups, and desserts.

## 🚀 Features
- **Responsive Design**: Optimized for all devices (Mobile, Tablet, Desktop).
- **Interactive UI**: Animated sections and smooth transitions.
- **Google Maps Integration**: Live location embed for the Nikol branch.
- **Accessibility Optimized**: Full compliance with `title`, `aria-label`, and `noopener` standards.
- **Performance**: Zero inline styles (centralized in `custom.css`) for faster loading.
